JSON的两种形式

分为两种形式JsonArray和JsonObject，其内部可以相互循环嵌套

  • JsonArray: [{"name": "John", "age": "10"}, {"name": "Jim", "age": "12"}]

最外面为中括号[]，内部可以为多个JsonObject

  • JsonObject: {"name": "leon", "age": "10", "hobbies": ["basketball", "tennis", "football"]}

最外面为大括号{}，内部为key-value形式，value依然可以嵌套array和object
